# [25083] 새싹
### [25083번: 새싹](https://www.acmicpc.net/problem/25083)
- 해설
    
    ```python
    새싹을 출력하면 되는데… 
    우리가 print로 출력할 때 ‘’ , 또는 “” 를 이용한다. 
    출력을 해야하는 곳에 ‘ 또는 “ 이 있다면 해당 문자를 출력할 것으로 보지 않고
    코드로 인식하기 때문에 바로 출력하기 위해서는 \ 를 붙여 주어야 한다.
    ```
- 답안
    
    ```python
    if __name__ == '__main__':
        print('         ,r\'"7')
        print('r`-_   ,\'  ,/')
        print(' \. \". L_r\'')
        print('   `~\/')
        print('      |')
        print('      |')
    ```
