# [10807] 개수 세기
### [10807번: 개수 세기](https://www.acmicpc.net/problem/10807)
- 해설
    
    ```python
    
    파이썬에서 리스트의 특정 값이 몇개가 있는지 체크하기 위해서 count를 사용할 수 있습니다.
    nums.count(V)
    
    이렇게 하면 nums에 V가 몇개인지 나오기 때문에 바로 출력하면 됩니다.
    ```
- 답안
    
    ```python
    if __name__ == '__main__':
        N = int(input())
        nums = list(map(int, input().split()))
        V = int(input())
        print(nums.count(V))
    ```
