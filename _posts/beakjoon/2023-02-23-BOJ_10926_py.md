# [10926] ??!
### [10926번: ??!](https://www.acmicpc.net/problem/10926)
입력 받은 것 뒤에 ??! 을 붙여서 출력하면 된다.
- 해설
    
    ```python
    **입력**
    따로 int 등의 자료형을 붙이지 않으면 str(string)형태로 들어간다.
    a = input()
    
    출력
    어떤 변수 뒤에 특정 문자를 붙이려면 +'' 로 붙이기가 가능하다.
    print(a+'??!')
    ```
- 답안
    
    ```python
    if __name__ == '__main__':
        a = input()
        print(a+'??!')
    ```
