# [1427] 소트인사이드
### [1427번: 소트인사이드](https://www.acmicpc.net/problem/1427)
- 해설
    
    ```python
    입력 받은 수를 내림차순으로 정렬하기만 하면 된다.
    list의 경우 sorted에 reverse 옵션을 True로 두면 내림차순이 된다.
    ```
- 답안
    
    ```python
    if __name__ == '__main__':
        N = int(input())
        nums = str(N)
        nums = ''.join(sorted(nums, reverse=True))
        print(nums)
    ```
